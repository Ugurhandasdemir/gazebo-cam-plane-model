
# plane_cam

Basit: PX4 SITL için uçak + kamera modeli.

Dosyalar:
- `model.config`
- `plane_cam.sdf`
- `plane_cam.sdf.jinja`

Hızlı kullanım:
- PX4 kök dizininde: `make px4_sitl gazebo`
- Gazebo'da Insert ile modeli ekleyin veya SDF'yi spawn edin.

Not: Detaylı talimat istemezseniz bu kadar yeterlidir.
